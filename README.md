***English:***

Code for simulating bank transactions via terminal using SQLite3 for data persistence. The system uses parameterized queries to protect the database against SQL Injection attacks and implements transaction control with commit and rollback to ensure money is only transferred if all security steps are successfully validated.

______________________________

***Português:***

Código para simular transações bancárias via terminal usando SQLite3 para persistência de dados. O sistema utiliza queries parametrizadas para proteger o banco contra ataques de SQL Injection e implementa controle de transações com commit e rollback para garantir que o dinheiro só seja transferido se todas as etapas de segurança forem validadas com sucesso.
